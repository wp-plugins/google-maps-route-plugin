Google Maps Route is an open source a solution built for Wordpress to display several locations along a route on Google Maps.

== Description ==

Google Maps Route is open source software that enables you to easily insert Google Maps, along with a route for your travel, tour, etc in your WordPress website. We have build very easy to use dashboard options to help add locations to maps directly. You will only need to have Google Maps API key which you can get [here](http://www.google.com/apis/maps/signup.html "Sign Up for the Google Maps API").

Google Maps Route can be used to place multiple locations and form  a route on Google Maps with those locations. You can place location addresses, descriptions and color for the route easily through Dashboard. We have provided the option of using Shortcode to easily embed the map in content and you can also show Maps via Widgets.

= Main Features: =

* Easily manage locations through Dashboard.
* Easily change the color, transparency and width of Route through Dashboard.
* Show Map in your Post or Pages by simply adding Shortcode `[route]`.
* Show Map via Widget.

Google Maps Route Plugin Documentation: [Google Maps Route Plugin Documentation](http://netmadeez.com/blog/google-maps-route-plugin/ "Google Maps Route Plugin Documentation")

== Installation ==

1. Upload the Google Maps Route folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Create a Google Maps API key [here](http://www.google.com/apis/maps/signup.html "Sign Up for the Google Maps API").
4. Put your Google Maps API key in Map Routes > Settings and save it.
5. You can directly start using it by either using shortcode `[route]` or widget.
6. After this, visit the [Google Maps Route Plugin Documentation](http://netmadeez.com/blog/google-maps-route-plugin/ "Google Maps Route Plugin Documentation")

== Frequently Asked Questions ==

= Where can I get a Google Maps API Key ? =

[http://code.google.com/apis/maps/signup.html](http://www.google.com/apis/maps/signup.html "Sign Up for the Google Maps API")

= What unit do you expect the map width and height to be in ? =

Pixels are used to define the dimensions.

= How can I insert Google Maps Route into my post/page ? =

By simply adding Shortcode `[route]` in your post's /page's content.

= How will the route show on the Google maps ? =

Once you add a list of locations from Map Routes > Add New Location, the route will be automatically be mapped. You only have to put the locations address and optionally description.

= How do I define all the locations for my Google Maps Route ? =

Through `Map Routes > Add New Location` you can “Add New Location” to the list of locations and view all your locations in Map Routes > List Locations.
